# Urban Citizen Feature Module

This module encompasses features built for the Urban Citizen view, mapping transit and local air quality telemetry.

## 📂 Folders

- **`components/`**: Urban transit recommendation widgets, air quality indexing details, and 2-layer Google Maps overlay.
- **`hooks/`**: Hooks to fetch transit options and check real-time air quality ratings.
- **`services/`**: API adapters for Sentinel-5P gas spectrometry analysis and routing services.
- **`types/`**: Interfaces representing transit modes, geolocations, and AQI indices.
- **`utils/`**: Utilities for calculating carbon emissions based on travel distance and modes.
