# Carbon Marketplace Feature Module

This module manages the carbon credit marketplace, user purchases, verification records, and transaction ledgers.

## 📂 Folders

- **`components/`**: Clean transactions dashboard, carbon credit calculator lists, and active sandbox Google Pay module button interface.
- **`hooks/`**: Custom hooks for executing credit purchases, monitoring transaction state, and fetching listings.
- **`services/`**: API wrapper client integrations for Google Pay sandbox and credit verification ledgers.
- **`types/`**: Types defining carbon offset credits, purchase logs, invoice states, and user transaction models.
- **`utils/`**: Utilities for parsing credit values, currency calculations, and formatting transaction IDs.
