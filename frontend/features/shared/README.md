# Shared Feature Module

This module manages general-purpose visual layout components, utility helpers, and configurations reused by other domain features.

## 📂 Folders

- **`components/`**: Layout wrappers, core map wrappers, unified sidebar components, dashboard layouts, and organic color telemetry cards.
- **`hooks/`**: Global application UI hooks (e.g. active view tabs, mobile sidebars toggle).
- **`services/`**: Standard telemetry feed listeners and Google Maps loader wrappers.
- **`types/`**: Shared TS typings, telemetry statuses, and general layout types.
- **`utils/`**: Shared mathematical helpers, accessibility helpers, and formatters.
