# Knowledge Base: Visual System & Brand Manual
## 🌿 Design Principles
- Style: Organic, minimal, calm. Inspired by Ecosia and National Geographic.
- Layout Rule: Maximize white space. Keep visual interfaces texturing flat and light to reduce user tracking fatigue.
## 🎨 Strict Color Token Matrix
- Forest Green (Primary): `#2E7D32`
- Leaf Green (Secondary): `#66BB6A`
- Earth Brown (Particulates): `#8D6E63`
- Sky Blue (Cool Canopy): `#81D4FA`
- Pale Sage Background: `#F1F8E9`
- UI Containers: `#FFFFFF`
- Editorial Font Color: `#1B5E20`
- Structural Borders: `#DCEED2`
