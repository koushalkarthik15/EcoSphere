# Backend Business Services Layer

Core integration logic and data fetchers:
- **`google_maps.py`**: Interacts with the Geocoding API.
- **`copernicus.py`**: Interacts with Copernicus Sentinel Hub.
- **`nasa_firms.py`**: Interacts with the active fire coordinate system.
- **`carbon_math.py`**: Implements equations for commutation offsets, soil carbon indices, and methane multipliers.
