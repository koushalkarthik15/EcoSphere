# API Routers Layer

FastAPI APIRouter definitions, grouped by feature domains:
- `/api/v1/urban`: Transit recommendation routing.
- `/api/v1/farmer`: Agricultural Sentinel-2 NDVI matrix maps and Google Sheets exports.
- `/api/v1/industry`: Facility geocoding bounding box lookups.
- `/api/v1/marketplace`: Carbon credit checkout processing.
- `/api/v1/auth`: Google identity JWT verification endpoints.
